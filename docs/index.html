Google Health MCP

Local-first MCP server for Google Health API v4. Tokens stay on your machine; agents get read-only tools for identity, profile, settings, data points, reconcile, daily rollups and summaries.

npx -y google-health-mcp-unofficial setup
npx -y google-health-mcp-unofficial auth
npx -y google-health-mcp-unofficial doctor

Google recommends waiting until the end of May 2026 before stable public launch because Google Health API v4 may still change.
